# Project to develop a website to serve an airline
Project of Web and application programming Subject

## Installation
- Install XAMPP at: [here](https://www.apachefriends.org/download.html)
- Copy the entire project into the `htdocs` folder. Usually the path will be `C:\xampp\htdocs`

## Configuration
- In the `database` folder of project, there is a file named `air_ticket_management.sql`
- Start Apache and MySQL in xampp control panel
- Click on `Import` and point to the `air_ticket_management.sql` file above
Database configuration:
- Click on "User accounts", get host, username, password information
- Go to project, `model/connection.php`, configure according to the information above

## Run project
- Open xampp control panel and start Apache and MySQL, the project will automatically load into the `index.php` file which is the home page of the website.
